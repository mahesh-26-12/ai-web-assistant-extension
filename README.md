# 🤖 🚀 AI Web Assistant Chrome Extension
## 🚀 Overview
🚀 AI Web Assistant Chrome Extension is a Chrome Extension that helps users summarize web pages, generate questions, and ask queries based on page content using AI.

## ✨ Features
- 📄 Summarize any webpage in seconds
- ❓ Generate important questions
- 💬 Ask questions based on page content
- 📋 Copy results instantly
- ⚡ Fast and lightweight

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Chrome Extensions API
- OpenRouter API


## 🔑 API Setup

1. Get your API key from https://openrouter.ai
2. Open popup.js
3. Replace:

   const API_KEY = "YOUR_API_KEY_HERE";

   with your actual API key

## ⚙️ Installation

1. Download or clone this repository
2. Open Chrome and go to:
   chrome://extensions
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select the project folder

## 🔑 Setup

- Get API key from https://openrouter.ai
- Replace API key in `popup.js`

## 📸 Usage

1. Open any webpage
2. Click the extension
3. Use:
   - Summarize
   - Generate Questions
   - Ask questions
